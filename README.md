# Digital Literacy Learning Platform

A frontend-only React application for learning digital literacy skills, deployable on GitHub Pages.

## Features

- User registration and login
- Interactive lessons
- Homework assignments
- Quizzes with scoring
- Typing practice with WPM and accuracy tracking
- Progress tracking and achievements
- Responsive design with accessibility features

## Installation

1. Clone the repository
2. Run `npm install` to install dependencies

## Running Locally

Run `npm start` to start the development server.

## Building for Production

Run `npm run build` to build the application for production.

## Deployment to GitHub Pages

1. Update the `homepage` field in `package.json` with your GitHub repository URL
2. Run `npm run deploy` to build and deploy to GitHub Pages

## Technologies Used

- React
- TypeScript
- Tailwind CSS
- Redux Toolkit
- React Router
